<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pedrol33/chat_app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app and view at localhost://3000
```sh
npm start
```
